# Задание 2.1. API-тесты

# Тест-сьют "Создание объявления"

## Общие предусловия:
1. Доступен хост `https://qa-internship.avito.com`.
2. Для позитивных и части негативных сценариев используется базовое валидное тело запроса.
3. При необходимости генерируется уникальный `sellerID` в диапазоне `111111–999999`.

## Базовое валидное тело запроса:
```json
{
  "sellerID": 367432,
  "name": "Test item",
  "price": 1000,
  "statistics": {
    "likes": 10,
    "viewCount": 100,
    "contacts": 5
  }
}
```

## Тест-кейс 1: Успешное создание объявления с валидными данными
### Цель:
Проверить, что API успешно создает объявление при передаче корректного тела запроса.

### Шаги:
1. Подставить уникальный `sellerID` в базовое валидное тело запроса.
2. Отправить `POST /api/1/item`.
3. Проверить статус-код ответа.
4. Проверить тело ответа.

### Ожидаемый результат:
1. Возвращается статус-код `200`.
2. В ответе присутствует уникальный `id`.
3. В ответе присутствуют поля:
   - `id`
   - `sellerId`
   - `name`
   - `price`
   - `statistics`
   - `createdAt`
4. Значения `name`, `price`, `statistics`, `sellerId` соответствуют отправленным данным.
5. Объявление создается
