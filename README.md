# Тестовое задание для Avito, QA-engineering.

## Структура проекта

```text
avito-qa
├── Task1 - папка 1 задания
│   └── README.md - описание задания 1
├── Task2 - папка 2.1 задания
│   ├── src
│   │   └── test/java/ru/avito/qa/api
│   │       ├── cases - data provider и модели сценариев
│   │       ├── client - HTTP-клиент и модель ответа
│   │       ├── endpoints - классы для работы с API-ручками
│   │       ├── support - фабрики данных и общие assertions
│   │       └── tests - тест-классы
│   ├── BUGS.md - описание найденных багов
│   ├── TESTCASES.MD - тест-кейсы к заданию 2.1
│   ├── README.md - инструкция по запуску Task2
│   └── pom.xml - зависимости и конфигурация Maven
└── README.md - общее описание проекта
```
